What does your Claude usage actually cost?

A cost breakdown of your Claude Code, Claude Desktop, Claude Cowork, and Agent-SDK usage, by day, model, surface, and session.

🔒 Local-only. Your transcripts are read and analysed entirely inside this page. No upload, no server, no network, no telemetry. Close the tab and nothing persists.

Which folder do I pick?
Everything under the folder is scanned; you can pick more than once to add sources.
→ ~/.claude  terminal + Desktop + Agent-SDK usage (all platforms)
→ ~/Library/Application Support/Claude  adds Claude Cowork usage + session titles (macOS)
macOS note: both folders start with a dot or live under a hidden ~/Library, so they're hidden in the file dialog. Press ⌘⇧. (Command-Shift-Period) in the picker to reveal them. To capture everything including Cowork, add both.
Prefer a terminal? Run python3 etl.py once and serve this folder over HTTP for an instant, refreshable dashboard. See the README.
